# DnD Sheet Translator

Tool zum Importieren englischer D&D-Charakterbögen und Exportieren in deutsche Charakterbogen-Vorlagen.

## Ziele

- Roll20-PDF importieren
- Charakterdaten zuverlässig erkennen
- deutsche Begriffe verwenden
- deutsche ausfüllbare PDF erzeugen
- Debug- und Korrekturmodus für OCR-Zonen
